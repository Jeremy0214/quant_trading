# 台股個股評估與買賣訊號預測系統

接收台灣股票代號，自動抓取財務與歷史股價資料，計算基本面與技術面指標，輸出含「買賣建議」與「信心指數」的評估報告。

---

## 專案結構

```
stock_forecast/
├── main.py                      # 程式入口
├── config.py                    # 評分門檻與指標參數
├── requirements.txt             # 套件依賴
└── modules/
    ├── data_fetcher.py          # 資料抓取（yfinance + MOPS）
    ├── fundamental_analyzer.py  # 基本面評分模組
    ├── technical_analyzer.py    # 技術面評分模組
    └── report_generator.py      # 報告格式化輸出
```

---

## 安裝

**環境需求：** Python 3.10+

```bash
pip install -r requirements.txt
```

---

## 使用方式

```bash
# 分析單支股票
python main.py 2330

# 批次分析多支股票
python main.py 2330 2412 6505

# 互動模式（執行後手動輸入）
python main.py
```

## 評分邏輯

系統總分 **100 分** = 基本面（50 分）+ 技術面（50 分）

### 基本面模組（50 分）

| 指標 | 滿分 | 計分規則 |
|------|------|----------|
| 營收成長性 | 15 | 月 YoY > 10% 且累計 YoY > 10% → +15；月 YoY > 0% → +5；衰退 → 0 |
| EPS 獲利能力 | 15 | EPS 年增或歷史新高 → +15；正值穩定 → +10；負值 → 0 |
| 本益比 P/E | 10 | P/E < 15 → +10；15~20 → +5；20~25 → +3；> 25 → 0 |
| ROE 效益 | 10 | ROE > 15% → +10；10%~15% → +5；< 10% → 0 |

> **資料來源：** EPS、P/E、ROE 來自 yfinance；月營收 YoY 來自 MOPS（公開資訊觀測站）。

### 技術面模組（50 分）

| 指標 | 滿分 | 計分規則 |
|------|------|----------|
| 均線趨勢 MA | 20 | 多頭排列且均線全上 → +20；多頭排列部分偏平 → +15；站上月線 → +10；跌破 → 0 |
| MACD & RSI | 15 | RSI 40~70 且 MACD 黃金交叉/轉正 → +15；RSI > 80 或死亡交叉 → 0 |
| 量價關係 | 15 | 量增價漲且突破高點 → +15；量增價漲 → +10；量平 → +5；**量增價跌 → -10（倒扣）** |

> 技術指標（MA5/MA20/MA60、RSI-14、MACD 12/26/9）均由歷史股價自行計算，不依賴外部技術分析套件。

### 買賣建議判定

| 總分 | 建議 | 信心指數 |
|------|------|----------|
| 80 ~ 100 | 🔴 強烈買入 Strong Buy | 高 |
| 60 ~ 79 | 🟡 適合買入 / 偏多看待 Buy | 中等 |
| 40 ~ 59 | ⚪ 觀望 / 中立 Hold | 低 |
| 20 ~ 39 | 🟡 建議減碼 / 偏空看待 Sell | 中等 |
| 0 ~ 19 | 🔴 強烈賣出 / 避開 Strong Sell | 高 |

## 資料來源說明

| 資料 | 來源 | 備註 |
|------|------|------|
| 歷史股價（1 年） | [yfinance](https://github.com/ranaroussi/yfinance) | 自動嘗試 `.TW`（上市）、`.TWO`（上櫃） |
| EPS、P/E、ROE | yfinance `ticker.info` | 部分小型股資料可能缺失 |
| 月營收 YoY | [MOPS 公開資訊觀測站](https://mops.twse.com.tw/) | 網路不通或 MOPS 維護時會顯示「資料無法取得」 |

---

## 注意事項

- **MOPS 月營收**：網站偶有防爬蟲機制，若顯示「MOPS 查詢失敗」屬正常現象，該項得 0 分並標示跳過。
- **投資免責聲明**：本系統輸出僅供參考，不構成任何投資建議。投資有風險，請自行評估後做出決策。
